Project Overview

난독증 아동이 그린 그림을 바탕으로 ControlNet을 활용해 동화의 삽화를 생성하고, OpenAI와의 대화를 통해 아이가 주도적으로 스토리를 만들어갑니다. 동화 제작 과정 속에 9종의 다양한 학습 게임이 단계별로 자연스럽게 배치되어 있어, 아이가 이야기 속에 몰입한 채 쉽고 재미있게 학습에 참여할 수 있도록 설계했습니다.

    팀원들의 신뢰와 지지를 바탕으로 성향과 역량을 파악해 역할을 분배하고, 항상 밝고 긍정적인 분위기를 만들기 위해 노력했습니다. 팀원들로부터 "함께여서 즐거웠다"는 피드백을 받으며 리더로서 성장했습니다.

    HTTPS 인증서 문제로 배포가 여러 차례 지연되는 상황에서 서로의 의견을 존중하고 함께 자료를 찾으며 실험을 반복해 해결했습니다. 협업의 중요성과 팀 내 시너지가 기술적 문제 해결의 핵심임을 체감했습니다.

    타겟층이 아동이기 때문에 아이들의 눈높이에 맞춘 표현과 흐름을 찾는 것이 핵심 과제였습니다. "기술보다 중요한 것은 결국 누구를 위해 만드는 것"임을 몸소 깨달은 소중한 경험이었습니다.

    8명의 다양한 직군과 함께한 한 달간의 프로젝트에서 PM, 프론트엔드, 백엔드까지 전방위적으로 참여했습니다. 이를 통해 기술적 성장과 커뮤니케이션 능력, 문제 해결 능력을 키울 수 있었습니다.

Project Overview

이민자 및 외국인의 법률 접근성 향상을 위해 GPT-4와 RAG 기반 다국어 챗봇을 개발했습니다. 언어 장벽과 정보 부족의 문제를 해결하기 위해 자연어 상담, 자동 번역, 음성 입출력을 도입했으며, Azure AI 서비스와 오픈 소스 기술을 결합한 웹 기반 실용 서비스입니다.

🌐 Frontend
  • GPT-4 및 RAG 기반 법률 Q&A 챗봇의 프론트엔드 개발 주도
  • 다국어 UI 설계 및 사용자 인터페이스 구현
  • 대화 저장 기능 추가로 사용자 경험 향상
🎤 음성 기능
  • Web Speech API를 활용한 음성 인식 기능 구현
  • 다국어 텍스트-음성 변환(TTS) 처리
  • 음성 기능으로 접근성 및 사용자 편의성 향상
📊 발표 제작
  • 발표 자료 및 프로젝트 시연 영상 제작
  • 서비스 아키텍처 및 기능 설명 자료 작성
  • ⚠️ JavaScript 기반 구글 음성 인식 API를 통해 음성을 다국어로 번역하는 기능 작동 오류
    → 사용자 접근성과 직결된 핵심 기능으로 즉각적인 해결 필요
  • 🔧
    • 프론트·백엔드 팀원 3명과 실시간 코드 공유 및 다각도 원인 점검
    • 테스트 주도를 통한 브라우저별 오류 재현 및 테스트 케이스 기반 문제 해결
    • 실시간 결과 공유를 통한 협업 효율성 극대화
  • ✅ 사용자 음성 접근성을 확보 → 프로젝트 핵심 기능 완성

    프로젝트를 진행하면서 HTML, CSS, JavaScript를 본격적으로 다뤘습니다. 대학 시절 이후로 프론트엔드 기술에 대한 실력이 많이 미비한 상태에서 시작했지만, ChatGPT를 적극적으로 활용하여 구조를 빠르게 이해하고, 반복적인 구현 과정을 통해 자신감을 쌓을 수 있었습니다. 자동 생성된 코드에만 의존하지 않고 직접 디자인을 수정하고 사용자 경험을 개선하는 과정에서 실력을 크게 향상시켰습니다. 그 결과, 프로젝트 발표 후 "감각적이다." "디자인이 다른 팀들과 비교해 뛰어나다."라는 긍정적인 피드백을 받았습니다.

Project Overview

사용자의 나이·성별·여행 스타일 등 9가지 입력 데이터를 기반으로 AI가 맞춤형 여행 코스를 추천합니다. 카카오 지도 API를 활용해 추천 결과를 직관적으로 시각화하며, 아기자기한 감성의 UI와 맞춤형 콘텐츠 구성으로 누구나 쉽고 즐겁게 사용할 수 있도록 설계되었습니다.

🎨 UI/UX 디자인
  • 사용자 친화적인 인터페이스 설계 및 전체 사용자 플로우 구성
  • 나이/성향 기반 입력 폼 UI 디자인 및 반응형 구성
  • 여행 감성을 살린 아이콘·배경 직접 제작
🗣 발표 기획
  • 발표 스토리보드 및 흐름 전체 구성
  • 디자인 슬라이드 및 시연 영상 제작 주도

    여행의 감성을 살리기 위해, UI에 사용되는 아이콘과 배경 등을 직접 손으로 그려 제작했습니다. 저작권 이슈를 피하면서도 브랜드 무드에 어울리는 감각적인 시각 요소를 구현했고, “귀엽고 감각 있다”는 피드백을 팀 내외부에서 받을 정도로 디자인 자체로 긍정적인 인상을 남겼습니다.

    사용자가 여행 성향을 입력하는 로그인 후 메인 입력 화면을 중심으로, 나이, 성별, 여행 스타일 등 약 7가지 항목을 수집하는 폼 UI를 기획, 디자인, 구현까지 전담했습니다. 추천 여정의 첫 관문인 만큼, 입력 과정이 귀엽고 부드럽게 이어지도록 컬러와 애니메이션을 활용해 완성도 높은 사용자 경험을 설계했습니다.
